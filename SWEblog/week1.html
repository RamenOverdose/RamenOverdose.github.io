What's your pick-of-the-week or tip-of-the-week?

Golang has a built-in testing package which makes running tests very efficient and easy to understand. If you want to test a file named "hello.go", you can simply create a test file named "hello_test.go", and Golang will understand that this is a test file. Once you have written your tests, which must follow certain naming conventions, you can simply execute "go test" and the tests will run. On top of this, Golang can also provide you with a code coverage report that is displayed in an html page for easy reading. You can find more information here.
