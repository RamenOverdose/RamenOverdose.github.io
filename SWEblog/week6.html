What's your pick-of-the-week or tip-of-the-week?

I recently learned the mechanics behind Git's commit and push functionalities. When I commit a change, Git saves that update in my local repository. When I push, Git updates the remote repository with the latest commit that I made. Then, the head is updated to point to this latest commit. I had previously never visualized commits as a chain, with a head pointer that can move around. When I switch to a branch, the head pointer also moves to that new branch. Visualizing these pointers made Git more interesting to me.
