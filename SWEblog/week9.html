What's your pick-of-the-week or tip-of-the-week?

Our group had trouble with multiple CSS files affecting our project in this phase. One CSS file would sometimes override another CSS file, and we could not figure out why that was happening. Although we did not fully understand the issue. I found a "duct tape" fix that works alright. After a line of CSS, you can add "!important" before the semicolon, which allows this CSS command to be treated as a priority. For instance, "color: white !important;" will ensure that your font is white.
