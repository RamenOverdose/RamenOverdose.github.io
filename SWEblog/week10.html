What did you think of the talk by Dr. Rich and Dr. Cline?

I thought their talk was very interesting, as ethics is a topic I rarely think about. As programmers, it is so easy for us to get caught up in the code that we forget about the purpose of our work. It was refreshing to think about such ethical dilemmas, and it's important to remember that technology is both beneficial and dangerous.

What's your pick-of-the-week or tip-of-the-week?

AWS has a service called CodePipeline, which is a continuous integration and continuous delivery service that allows a project to be updated, built, and tested automatically before deployment. It may not be worth it for smaller projects, but can make the deployment process for larger projects much more efficient. You can define the build and testing stages as you wish, and there is an option to have the pipeline deploy automatically or have the user deploy manually.
