What's your pick-of-the-week or tip-of-the-week?

Homebrew is a package manager for MacOS and Linux. It makes installing missing packages very easy. Everything can be done through the command line. For instance, if you want to update git, you can simply enter "brew upgrade git" and git will update automatically. If you have MacOS, you should check it out. You can find more information here.
