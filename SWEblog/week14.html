What's your pick-of-the-week or tip-of-the-week?

When working with JavaScript state variables in React, sometimes they will not update before the page is loaded. I found a duct tape way to fix this, which forces the component to render. This is definitely not the "right" way to fix the problem, but can be useful in emergencies. "this.forceUpdate()" forces a rerender and will allow the state variable to be set.
